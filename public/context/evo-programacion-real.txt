PATRONES CLAVE DEL ESTILO EVO — RESUMEN PARA LA IA
================================================================

FORMATOS DE FUERZA (de más a menos frecuente):
1. E2MOM / E2:30 / E3:00 x N sets
2. EMOM ODD/EVEN (fuerza + complementario)
3. Every 1:30 for N sets (técnica compleja)
4. For Load EMOM (subir carga cada ronda)
5. Cluster Set con rest entre series
6. Biserie/Triserie 90''-3' entre rondas
7. Wave Loading (solo squat, deadlift, press pesado)

FORMATOS DE WOD (de más a menos frecuente):
1. AMRAP con intervalos dinámicos every X min
2. Partner IGYG con synchro para dar relevo
3. LADDER ascendente (+1 o +2 cada ronda)
4. AMRAPs ON/OFF (5' ON / 1:30 OFF x 3-4)
5. FOR TIME con pirámide (21-15-9 o 30-20-10-20-30)
6. EMOM largo metabólico (16'-20') con 4 estaciones
7. Squat Party / EMOM con penalización

ORGANIZACIÓN DE SALA (incluir siempre en el feedback):
- Cómo colocar el material antes de empezar
- Quién trabaja primero si hay menos material que personas
- Qué escribir en la pizarra (esquema WOD, pesos, tiempo)
- Organización por parejas o grupos desde el inicio

PESOS ORIENTATIVOS (siempre dar referencia concreta):
KB Swing Americano: Rx 28/20kg | Esc 24/16kg
KB Swing Ruso: Rx 24/20kg | Esc 20/16kg
Wall Ball: siempre 7kg
DB Snatch: Rx 22.5/15kg | Esc 17.5/12.5kg
Barra Press: indicar % del RM
Farmer Carry: discos pesados de agarre de pinza

BASICS — INSTRUCCIÓN FIJA:
Siempre pedir que apunten el peso conseguido en la App.
"Buscar peso con el que hacer X reps y apuntar en App."

SCORE EN PAREJAS/EQUIPOS:
Siempre definir el score antes de empezar.
Ejemplos: reps acumuladas, FOR TIME, rondas completas.
